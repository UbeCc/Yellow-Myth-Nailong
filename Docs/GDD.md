# Yellow Myth: Nailong — Game Design Document

## 核心概念

致敬《黑神话：悟空》的暗黑国风动作游戏，主角是一只原创的黄色小胖龙“奶龙”，手持金箍棒，挑战黑风山妖王。

## 玩法循环

1. 在破庙场景中移动、探索。
2. 遭遇 BOSS“黑风山·熊罴怪”。
3. 使用轻攻击、重攻击、闪避与 BOSS 战斗。
4. 击败 BOSS 或玩家血量归零判定胜负。

## 输入

| 输入 | 动作 |
|------|------|
| WASD | 移动 |
| 鼠标 | 视角 |
| 左键 | 轻攻击 |
| 右键 | 重攻击 |
| 空格 | 闪避 |
| Esc | 退出 |

## 角色

### 奶龙（主角）

- 造型：黄色圆胖身体、橙色小角、小翅膀、大尾巴。
- 武器：金箍棒。
- 能力：轻攻击、重攻击、闪避。

### 黑风山·熊罴怪（BOSS）

- 造型：暗色巨熊，手持狼牙棒，红眼。
- AI：追击、蓄力攻击、受击硬直。

## 画面氛围

- 低饱和、高对比度
- 体积雾与暖色夕阳光
- 暗角、微弱泛光
- 国风破庙场景

## 技术栈

- O3DE 26.05
- Atom Renderer
- Lua 脚本 / Script Canvas
- PhysX 5
- LyShine UI
